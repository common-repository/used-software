Plugin to display information about used softwares.

== Description ==

This plugin adds a custom Gutenberg block: "Software Information". This block allows you to display information about the software used on your website.

== Installation ==

Upload the plugin directory to the /wp-content/plugins/ directory or install the plugin directly through the WordPress plugin screen.
Activate the plugin through the 'Plugins' menu in WordPress.
== Usage ==

After installing and activating the plugin, a new block will be available:

Software Information:
Adds a table with information about the software used.
You can specify the software name, version, and a link to the software.
The block provides options to show/hide links and versions, as well as add/remove software entries.
== Frequently Asked Questions ==

= How do I add the block to my post or page? =

Edit a post or page.
Click the "+" icon to add a new block.
Search for "Software Information" and add the desired block.
Configure the block settings as needed and save your changes.
